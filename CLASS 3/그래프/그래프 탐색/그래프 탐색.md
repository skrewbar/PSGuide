# 격자 그래프
그래프 탐색 문제를 풀다 보면 격자를 탐색해야 할 때가 많은데, 아래처럼 dx, dy배열을 만들어서 탐색하면 가독성을 챙길 수 있습니다.
```python
def dfs(...):
    ...

    dx = [1, -1, 0, 0]
    dy = [0, 0, 1, -1]

    for k in range(4):
        nx = x + dx[k]
        ny = y + dy[k]
        if nx < 0 or nx >= n or ny < 0 or ny >= m:
            continue
    ...
```
