## PS란?

PS는 Problem Solving의 줄임말로, 프로그래밍 문제를 푸는 것을 의미해요.

이곳은 자료구조와 알고리즘을 공부하고, 프로그래밍 문제를 푸는 사람들이 백준을 통해 쉽게 입문할 수 있도록 하기 위해 만들어졌어요.  
PS를 할 때 도움이 되는 사이트인 [백준][1]과 [솔브드][2]에 대해 알려드릴게요.

# 목차
- 백준에서 문제 풀어보기
- 솔브드 연동하고 설정하기
